# プロジェクト・タイトル

これは**VSCode**の*Markdownプレビュー機能*を試すための、一般的なサンプルテキストです。



---

## 概要

このドキュメントは、マークダウンの主要な記法が一通り含まれています。見出し、リスト、コードブロック、テーブルなどがどのように表示されるかを確認するのに最適です。

### 主な特徴

- **見出し:** `#` の数でレベルが変わります。
- **リスト:** 順序なしリストと順序付きリストの両方を含みます。
- **コード:** シンタックスハイライトが適用されたコードブロック。

---

## セットアップ手順

1.  必要なファイルをダウンロードします。
2.  ターミナルで以下のコマンドを実行します。
    ```bash
    # 依存パッケージをインストール
    npm install
    ```
3.  次のコマンドでプログラムを起動します。
    ```javascript
    const app = require('./app');
    app.start();
    ```

> **注意:** 上記のコードはあくまで表示確認用のサンプルです。

## 設定項目

| パラメータ | 型      | 説明                     |
| :--------- | :------ | :----------------------- |
| `user`     | `String`  | ユーザー名を指定します。   |
| `port`     | `Number`  | 接続ポートを指定します。 |
| `isLive`   | `Boolean` | 本番モードで起動します。   |
